#  6 个月路线（每月一个主题）

---

##  第 1 个月：Linux + Git + CI/CD 基础骨架

**目标：不被工具绊倒**

你要会的：

* Linux：

  * systemd、端口、日志、权限
* Git：

  * rebase / tag / rollback
* CI/CD 概念：

  * pipeline / stage / job
  * CI vs CD

 实战任务：

* 用 GitLab / GitHub
* 写一个最简单 pipeline：

  ```text
  code → test → build
  ```

---

##  第 2 个月：Docker + 镜像交付

**目标：代码 → 镜像**

你要会的：

* Dockerfile
* 多阶段构建
* 镜像版本管理（tag）

 实战任务：

* Java / Node / Python 任一
* 构建镜像并推送到私有仓库
* 镜像 tag 不用 latest（很重要）

---

##  第 3 个月：Kubernetes 核心（CKA 主线）

**目标：你已经在这条路上了**

重点内容：

* Pod / Deployment / Service
* PV / PVC（你前面问的都对路）
* Ingress / Ingress Controller
* 滚动升级、回滚

 本月目标：

* **开始 CKA 系统刷实操**
* 能手写常见 YAML

---

##  第 4 个月：CI/CD + Kubernetes 联动

**目标：流水线自动发布**

你要会的：

* kubectl in pipeline
* kubeconfig / token
* helm 或 kustomize（二选一）

✅ 实战任务（非常重要）：

```
git push
 ↓
CI 构建镜像
 ↓
更新 deployment image
 ↓
滚动发布到 K8s
```

这是 DevOps 的**分水岭** 

---

##  第 5 个月：生产级能力（面试杀手锏）

**目标：你像个“真运维”**

你要补齐：

* 灰度发布
* 回滚策略
* Ingress 升级不中断
* readiness / liveness
* 资源限制 & HPA

✅ 实战任务：

* 模拟一次“失败发布”
* 5 分钟内回滚

---

##  第 6 个月：证书 + 项目包装

**目标：能投简历**

 证书建议：

* **CKA（首选）**
* 后续可加：

  * AWS DevOps Pro / AZ-400

 简历重点写：

* “搭建 CI/CD 流水线”
* “K8s 滚动升级、回滚”
* “Ingress 无感升级”

---

##  最小证书组合（性价比最高）

> **CKA + 一个完整 CI/CD 项目**

这比：

* Jenkins 证
* Docker 证
  更值钱。
